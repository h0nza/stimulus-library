## Purpose

A controller that adds the ability for Trix editors to format text as strikethrough elements.

Fire and forget. The only configuration this controller, is a rule in your CSS for how it should look.
Mount it on your `<trix-editor>` element, add a style rule for

```css
.trix-button.trix-button--icon.trix-button--icon-strikethrough::before {
  background-image: url("data:image/svg+xml;base64,...");
}
```
and you're good to go.

## Side Effects

The controller will inject a button into the Trix toolbar, and modify the Trix instance to be able to handle
strikethrough formatting.
